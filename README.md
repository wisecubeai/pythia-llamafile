# pythia-llamafile
Run your own OpenAI API locally. Tested to work on Mac OS X & Linux. 

:eyes: ***build***:
  - make sure you have docker installed
  - clone this repository
  - build the image
```
./build.sh
```

:eyes: ***run***:
```
./run.sh
```
:eyes: ***UI Test***:  
  - go to http://127.0.0.1:8080
  - say 'hello'! 

:eyes: ***API Test***:  
```
python -m client
```
